Decoding the Project


  • Self-Healing: Auto-detects link failures and reroutes traffic for zero downtime.
  • BGP / OSPF: Internal routing efficiency (OSPF) combined with external multi-network connections (BGP).
  • Traffic Shaping: Prioritizing bandwidth for critical applications by delaying non-essential data.

Simulation Methodology


  • Phase 1: Mock Labs
    Observed OSPF Link-State Advertisement (LSA) tables in isolated setups.
  • Phase 2: Enterprise Sandbox
    Transitioned to Cisco Modeling Labs (CML) hosted on VMware.
  • Why CML?
    Employs authentic Cisco IOS images for realistic protocol behavior testing.

Roadblock: EIGRP Failures

Routers refused to establish neighbor relationships despite active links.


  • Root Cause 1: Mismatched AS Numbers
    Virtual routers were misconfigured with different Autonomous System numbers.
  • Root Cause 2: Interface Suppression
    The passive-interface command was inadvertently blocking necessary hello packets.

The Road Ahead


  • Practical BGP: Establishing multi-AS neighbor formations in the simulator.
  • Packet Analysis: Studying exact BGP data structures via simulated captures.
  • Optimization: Deploying the dynamic traffic shaping policies.
